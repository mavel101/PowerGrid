Für eine Kompilierung müssen die Pakete installiert werden, die auf der Power Grid Github Seite (https://github.com/mrfil/PowerGrid) angegeben sind.
Insbesondere muss Armadillo in der Version 8.600.x installiert werden (oder 9.200.x - gibt es nur im Docker Container von PowerGrid).
Funktioniert auch mit ISMRMRD 1.4.2.
Cmake sollte mit den Optionen OPENACC_GPU=OFF und OPENACC_MP=ON/OFF ausgeführt werden. (kann auch in CMakeLists.txt geändert werden)

Für Kompilierung:
mkdir build
cd build
cmake ..
make
sudo make install
